## 🚀 快速启动

### 方式一：使用脚本（推荐）

```bash
# 1. 安装依赖
chmod +x scripts/*.sh
./scripts/install.sh

# 2. 启动MongoDB
brew services start mongodb-community  # macOS
# 或
sudo systemctl start mongod  # Linux

# 3. 启动应用
./scripts/start.sh
```

### 方式二：手动启动

```bash
# 1. 安装依赖
npm install

# 2. 启动MongoDB（同上）

# 3. 启动应用
npm run dev
```

### 访问地址
- 前端：http://localhost:5173
- 后端：http://localhost:3000/api
- 健康检查：http://localhost:3000/api/health
