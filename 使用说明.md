# Click Book 使用说明

## 📚 欢迎使用 Click Book

Click Book 是一个功能强大的3D翻页电子书在线制作平台，让你轻松创建专业的电子书作品。

## 🎯 主要功能

### 1. 用户系统
- 注册和登录账号
- 管理个人资料
- 会员等级系统（免费、高级、VIP）

### 2. 创建电子书
- 上传PDF文件，自动拆分为单页
- 批量上传图片，创建电子书
- 支持的格式：PDF、JPG、PNG

### 3. 编辑功能
- 添加和编辑文字
- 添加背景音乐
- 自由调整元素位置
- 丰富的样式设置

### 4. 作品管理
- 保存草稿
- 发布作品
- 生成分享链接
- 在线阅读

## 📖 详细使用教程

### 第一步：注册登录

1. 打开网站首页
2. 点击右上角"注册"按钮
3. 填写邮箱、用户名和密码
4. 点击"注册"完成账号创建
5. 自动登录进入系统

### 第二步：创建电子书

#### 方式一：上传PDF

1. 点击"开始制作"或"创建作品"按钮
2. 选择"单文件上传"
3. 拖拽或选择PDF文件
4. 等待上传和处理
5. 自动进入编辑器

#### 方式二：上传图片

1. 点击"开始制作"或"创建作品"按钮
2. 选择"图片上传"
3. 选择多张图片（可按住Ctrl/Cmd多选）
4. 等待上传和处理
5. 自动进入编辑器

### 第三步：编辑电子书

#### 编辑器界面介绍

```
┌─────────────────────────────────────────────────┐
│  标题编辑  |  页数显示  |  保存  |  发布与导出  │ 顶部栏
├──────┬──────────────────────────────┬───────────┤
│      │  [文本] [音乐]              │           │
│ 页   ├──────────────────────────────┤  属性     │
│ 面   │                              │  面板     │
│ 列   │       画布区域               │           │
│ 表   │       (预览和编辑)           │  - 字体   │
│      │                              │  - 颜色   │
│ 第1页│                              │  - 大小   │
│ 第2页│                              │  - 对齐   │
│ ...  │                              │  - 等等   │
└──────┴──────────────────────────────┴───────────┘
```

#### 添加文字

1. 点击顶部工具栏的"文本"按钮
2. 在页面上出现默认文本框
3. 双击文本框进入编辑模式
4. 输入你想要的文字
5. 点击文本框外部完成编辑

#### 设置文字样式

1. 点击选中文本框
2. 在右侧属性面板中调整：
   - **字体大小**：调整文字大小
   - **字体类型**：选择Arial、微软雅黑、宋体、黑体
   - **文本颜色**：点击颜色选择器
   - **字重**：选择正常或粗体
   - **对齐方式**：左对齐、居中、右对齐
   - **行高**：调整行间距
   - **字间距**：调整字符间距

#### 移动和调整文字

- **移动位置**：点击拖动文本框
- **调整大小**：拖动文本框边缘（开发中）
- **删除文字**：选中后点击"删除"按钮

#### 添加音乐

1. 点击顶部工具栏的"音乐"按钮
2. 在弹出的音乐库中选择：
   - **共享资源**：使用平台提供的默认音乐
   - **我的音乐**：上传自己的音乐文件
3. 点击"使用"按钮添加到页面

#### 设置音乐属性

1. 点击选中音乐图标
2. 在右侧属性面板中调整：
   - **播放模式**：
     - 播放一次：点击播放一次
     - 循环播放：循环播放音乐
     - 进入页面时播放：翻到此页自动播放
   - **图标大小**：调整音乐图标的大小
   - **图标颜色**：自定义图标背景色

#### 页面管理

- **切换页面**：点击左侧页面列表中的缩略图
- **添加页面**：点击页面列表底部的"添加新页"按钮（开发中）
- **删除页面**：选中页面后删除（开发中）
- **排序页面**：拖动页面缩略图调整顺序（开发中）

### 第四步：保存和发布

#### 保存草稿

1. 在顶部输入作品标题
2. 点击"保存"按钮
3. 作品自动保存到"我的作品"

#### 发布作品

1. 确保作品已保存
2. 点击"发布与导出"按钮
3. 系统生成唯一的分享链接
4. 可以复制链接分享给他人

### 第五步：管理作品

#### 查看我的作品

1. 点击导航栏的"我的作品"
2. 查看所有创建的电子书
3. 显示作品标题、页数、更新时间

#### 编辑作品

1. 在作品列表中找到要编辑的作品
2. 点击"编辑"按钮
3. 进入编辑器继续修改

#### 分享作品

1. 确保作品已发布
2. 点击"分享"按钮
3. 自动打开分享链接
4. 复制链接发送给朋友

#### 删除作品

1. 在作品列表中找到要删除的作品
2. 点击"删除"按钮（垃圾桶图标）
3. 确认删除操作

### 第六步：在线阅读

#### 查看已发布的作品

1. 点击分享链接
2. 进入阅读器界面
3. 可以看到：
   - 作品标题
   - 当前页码
   - 页面内容
   - 文字叠加
   - 音乐按钮

#### 阅读操作

- **翻页**：点击左右箭头按钮
- **播放音乐**：点击页面上的音乐图标
- **全屏**：点击右上角全屏按钮
- **查看页码**：顶部显示"第X页 / 共Y页"

## 💡 使用技巧

### 文字编辑技巧

1. **文字对齐**：使用居中对齐让标题更美观
2. **字体搭配**：标题用大号粗体，正文用小号正常字重
3. **颜色选择**：使用对比色让文字更清晰
4. **行间距**：适当增加行高让文字更易读

### 音乐使用技巧

1. **背景音乐**：选择舒缓的音乐作为背景
2. **图标位置**：放在不影响阅读的角落
3. **图标颜色**：选择与页面协调的颜色
4. **播放模式**：重要页面可设置自动播放

### 作品制作技巧

1. **封面设计**：第一页作为封面，添加标题和作者信息
2. **目录页**：第二页可以制作目录
3. **内容排版**：保持统一的风格和间距
4. **音乐搭配**：不同章节可以用不同的音乐
5. **结尾页**：最后一页添加结束语或联系方式

## ❓ 常见问题

### Q1: 支持哪些文件格式？

**答**：
- 文档：PDF
- 图片：JPG、PNG
- 音频：MP3、WAV、OGG

### Q2: 文件大小有限制吗？

**答**：是的，单个文件最大支持50MB。

### Q3: 如何删除添加的文字或音乐？

**答**：
1. 点击选中要删除的元素
2. 在右侧属性面板点击"删除"按钮

### Q4: 可以修改已发布的作品吗？

**答**：可以。在"我的作品"中点击"编辑"按钮重新编辑，然后再次发布即可。

### Q5: 分享链接会过期吗？

**答**：不会，分享链接永久有效。

### Q6: 可以下载作品吗？

**答**：当前版本支持在线查看，下载功能正在开发中。

### Q7: 支持协作编辑吗？

**答**：当前版本不支持，未来版本将支持多人协作。

### Q8: 如何修改密码？

**答**：用户资料管理功能正在开发中。

### Q9: 忘记密码怎么办？

**答**：密码重置功能正在开发中。

### Q10: 作品数量有限制吗？

**答**：免费用户可以创建无限作品，部分高级功能需要会员。

## 🎨 设计建议

### 颜色搭配

- **商务风格**：深蓝 + 白色
- **温馨风格**：浅黄 + 棕色
- **清新风格**：绿色 + 白色
- **活泼风格**：橙色 + 黄色

### 字体选择

- **标题**：黑体 + 大号 + 粗体
- **正文**：微软雅黑 + 中号 + 正常
- **注释**：宋体 + 小号 + 正常

### 布局建议

- 保持页面留白，不要太拥挤
- 文字居中或左对齐，保持一致
- 图片和文字比例协调
- 重要内容用颜色突出

## 📱 平台支持

- ✅ 电脑（推荐）
- ✅ 平板
- ✅ 手机

**最佳体验**：在电脑上使用Chrome、Firefox或Safari浏览器

## 🔒 隐私和安全

- 所有密码加密存储
- 作品默认私密，只有发布后才能通过链接访问
- 未发布的作品只有作者本人可以看到
